Memo test

A simple tool to practice memory sports or BLD memo, inspired by this post.
5BLD is approximately 43 letter-pairs (86 letters), 4BLD ≅ 23 letter-pairs. For multi, it's about 10-12 images per cube, so setting up 12 items per row will be good for practicing MBLD.
This app is intended to be as simple as possible, without a "concentration→memorization→recall→review" cycle. If you want to play this scenario however, you can use separate timer to measure memo time, then type memorized letters into the left textarea of the diffchecker and the original generated symbols (check the browser console) into the right one.
Have fun!
